**AWS Budgets**: Setting a monthly cost budget with a fixed target amount to track all costs associated with your account. You can choose to be alerted for both actual (after accruing) and forecasted (before accruing) spends


**AWS Cost explorer**: AWS Cost Explorer is for detailed cost analysis and visibility, providing historical and forecastable cost and usage data through interactive visualizations to help understand spending patterns and identify optimization opportunities



**AWS cost analyser**

A fully-managed service for file transfers into and out of Amazon S3 or

**AWS app runner:**
Fully managed service that makes it easy to deploy web
Applications(containers) and APIs at scale.

**AWS App2Container:**
CLI tool for migrating and modernizing Java and .NET web apps into
Docker Containers. Deploy to ECS, EKS, or App Runner

**Amazon Rekognition** – Content Moderation

**Amazon Comprehend Medical**: For Natural Language Processing – NLP. Uses NLP to detect Protected Health Information (PHI)
**Amazon Kendra** - Fully managed document search service powered by Machine Learning
**Amazon Personalize** - ML-service to build apps with real-time personalized recommendations
**Amazon Textract** - Automatically extracts text, handwriting, and data from any scanned

**CloudWatch Logs Insights**: find a specific IP inside a log, count occurrences of “ERROR” in your logs
**CloudWatch Contributor Insights**: Analyze log data and create time series that display contributor data.
• See metrics about the top-N contributors
• The total number of unique contributors, and their usage

**AWS IAM Identity Center**: One login (single sign-on) for all your AWS accounts in AWS Organizations. 
**Permission Sets** – a collection of one or more IAM Policies assigned to users and groups to define AWS access.

**AWS Control Tower:** Easy way to set up and govern a secure and compliant multi-account AWS environment based on best practices
**AWS Control Tower – Guardrails**: Preventive Guardrail – using SCPs (e.g., Restrict Regions across all your accounts)
**Detective Guardrail** – using AWS Config (e.g., identify untagged resources)

**Amazon Security Lake**: is a fully managed, purpose-built service designed to automatically collect, normalize, and centralize security-related data from various AWS accounts, Regions, services, and even third-party sources. 



**VPC **– Traffic Mirroring: Allows you to capture and inspect network traffic in your VPC

**AWS Backup:** Centrally manage and automate backups across AWS services. Supports cross-region backups. Supports cross-account backups. You create backup policies known as Backup Plans. AWS Backup Vault Lock

**AWS Batch** supports multi-node parallel jobs, which enables you to run single
jobs that span multiple EC2 instances. Batch will dynamically launch EC2 instances or Spot Instances. Any runtime as long as it’s packaged as a Docker image

**AWS ParallelCluster**: Open-source cluster management tool to deploy HPC on AWS. Ability to enable EFA on the cluster (improves network performance)

**CloudFormation** – Service Role: User must have iam:PassRole permissions

**Amazon Pinpoint:** Supports email, SMS, push, voice, and in-app messaging

**AWS Cost Anomaly Detection:** Continuously monitor your cost and usage using ML to detect unusual spends

**Amazon AppFlow: ** Fully managed integration service that enables you to securely transfer data between Software-as-a-Service (SaaS) applications and AWS

**AWS Amplify:** A set of tools and services that helps you develop and deploy scalable full stack web and mobile
Applications

**Instance Scheduler on AWS:** Automatically start/stop your AWS services to reduce costs (up to 70%). Supports cross-account and cross-region resources

**AWS Well-Architected Tool:** Free tool to review your architectures against the 6 pillars Well-Architected
Framework and adopt architectural best practices
**Trusted Advisor: **Analyze your AWS accounts and provides recommendation on 6 categories
